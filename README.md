# Blood Infection Diagnosis Expert System

## Overview
A rule-based Artificial Intelligence system that assists in the preliminary diagnosis of blood infections based on user symptoms. The system uses a knowledge base and an inference engine to provide possible diagnoses using predefined medical rules.

> **Disclaimer:** This project is developed for educational purposes only and is not intended to replace professional medical advice.

## Features
- Rule-based diagnosis using IF-THEN rules
- Interactive symptom input
- Knowledge-based reasoning
- Diagnostic recommendations
- User-friendly interface

## Technologies Used
- Python
- Rule-Based Expert System
- Knowledge Base
- Inference Engine

## How It Works
1. User enters symptoms.
2. The inference engine compares symptoms with the knowledge base.
3. Matching rules are executed.
4. The system displays a possible diagnosis and recommendation.

## Example Rule
IF Fever AND Chills AND High WBC Count  
THEN Possible Blood Infection
